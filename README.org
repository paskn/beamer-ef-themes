Beamer color themes based on Protesilaos Stavrou's [[https://github.com/protesilaos/ef-themes][ef-themes]].

Currently, this includes:

- =ef-reverie= - =beamercolorthemeefreverie.sty=

* How to use

1. Copy a file the color theme (=*.sty=) into the same directory
   storing your slides.
2. In the slides file, add:

   #+begin_src shell
   \usecolortheme{efreverie}
#+end_src

** =org-mode= example

#+begin_src org-mode
#+options: H:2
#+latex_class: beamer
#+columns: %45ITEM %10BEAMER_env(Env) %10BEAMER_act(Act) %4BEAMER_col(Col) %8BEAMER_opt(Opt)
#+beamer_theme: default
#+beamer_color_theme: efreverie
#+beamer_font_theme:
#+beamer_inner_theme:
#+beamer_outer_theme:
#+beamer_header:
#+end_src
